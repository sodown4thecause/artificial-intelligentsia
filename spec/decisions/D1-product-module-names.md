# D1 — Product and Primary Assistant Module Names

- **Status:** CLOSED
- **Date:** 2026-07-20

## Decision

The product name is **Creature OS**. Its primary assistant module is **Go Agent**.

## Rationale

These names provide a stable product-level identity while giving the primary assistant a distinct, concise module name for product, engineering, and documentation use.

## Naming table

| Scope | Locked name | Usage |
|---|---|---|
| Product | Creature OS | Product, repository, and release-facing references |
| Primary assistant module | Go Agent | Primary assistant module references |

## Consequences

- New user-facing and decision documentation must use the locked names.
- This decision resolves Gate 1 naming only; it does not select a desktop shell or establish installer, launch, signing, notarization, or keyring evidence.
